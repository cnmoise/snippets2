  • End tags are required, else your code will be a mess.
  • Empty elements are signified with a closing slash.
  • Attribute values must be contained in quotation marks.
  • Attribute-value pairs must be fully written.

Useful Definitions

Element
A component of a web page defined and contained by a pair of identifiers known as tags. Single tag elements are known as empty elements and have no contents
Attribute
Allows developers to add more information to an element. They are usually placed in the start tag in the format name=value
DTD
(Document Type Definition) - A line of code which tells a file the elements, attributes, and syntax rules it must abide to. Useful for dealing with older, less well maintained websites
URL
(Uniform Resource Locator) - The web address which when entered into the web address bar will connect the user to a web server. The web server will send the browser a file to download which is stored on the hardware
W3C
(World Wide Web Consortium) - Founded by Tim Burners Lee in 1994, this organization sets standards and spearheads development for web technologies.
